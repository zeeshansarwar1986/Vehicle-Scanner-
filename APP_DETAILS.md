# Vehicle Scanner AI - Project Details

## App Description (English)

Vehicle Scanner AI is an advanced mobile application that leverages cutting-edge YOLOv8 models for real-time vehicle and license plate detection. Integrated with Google ML Kit for OCR (Optical Character Recognition), it automatically reads and logs vehicle plate numbers with precise GPS location and timestamps. This app is designed for security personnel, parking management systems, and law enforcement to streamline vehicle monitoring.
